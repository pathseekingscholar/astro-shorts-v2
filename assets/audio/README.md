# Audio Library

Royalty-free ambient music for Shorts.

## Tracks
The renderer will randomly select from available .mp3 files in this folder.

## Adding Music
1. Download royalty-free tracks from:
   - YouTube Audio Library: https://studio.youtube.com/channel/UC/music
   - Pixabay Music: https://pixabay.com/music/
   - Free Music Archive: https://freemusicarchive.org/

2. Choose ambient/space/cinematic tracks (20-60 seconds ideal)

3. Upload .mp3 files to this folder

## License
Only use tracks that are:
